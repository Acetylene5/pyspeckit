A spectroscopic toolkit for analysis of astronomical spectra. Targeted at optical, infrared, and radio spectroscopy.

Clone command:
hg clone https://bitbucket.org/pyspeckit/pyspeckit.bitbucket.org pyspeckit
If you don't have hg, you can grab the latest tarball.

Sphinx-generated Documentation

Installation instructions:
  • hg clone https://bitbucket.org/pyspeckit/pyspeckit.bitbucket.org pyspeckit
  • cd pyspeckit
  • python setup.py install

Note the basic requirements:
  • numpy
  • matplotlib
  • mpfit
  • scipy (optional, only required for RADEX grid interpolation)

And at least one of:
  • pyfits
  • atpy
  • asciitable [github]
  • hdf5 [pytables]
