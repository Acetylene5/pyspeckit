A spectroscopic toolkit for analysis of astronomical spectra. Targeted at optical, infrared, and radio spectroscopy.
Bitbucket Repository

Clone command:
hg clone https://pyspeckit@bitbucket.org/pyspeckit/pyspeckit.bitbucket.org pyspeckit

Installation instructions:
  • hg clone https://pyspeckit@bitbucket.org/pyspeckit/pyspeckit.bitbucket.org pyspeckit
  • cd pyspeckit
  • python setup.py install
